# 调试器：gdb
* :l or list          显示代码
* :<enter>            重复上一条命令
* :break 16           在第16行设置断点
* :break func         在函数入口处设置断点
* :info break         查看断点信息
* :r or run           运行程序
* :n or next          单步执行
* :c or continue      继续执行
* :p or print i       打印变量i的值
* :finish             结束程序
* :info register      查看寄存器的值
